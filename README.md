## :biking_man: PBike

Project for the MPS discipline with the objective of putting into practice crucial definitions in software engineering.
Concepts covered:

- Design patterns;
- Code smells;
- Hexagonal architecture;
- Exceptions;
- Inheritance;
- SOLID.
